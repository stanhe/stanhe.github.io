windows data recover

发表于 2022-06-16 更新于 2024-06-23

windows上数据被误删，如何快速恢复，先说结论：R-Studio

起因

某天我媳妇小手一抖把她的日报文件删除了，shift+Delete那种，没有回退或备份，欲哭无泪，赶紧找IT帮忙，IT也不会，就上某宝，找了个10块钱的，人家知道她挺急的，开口就是3个半毛爷爷。正应了地精那句话：“时间就是金钱，我的朋友”。

我告诉她，她可能被宰猴子了，猴哥说：“那也没办法，确实挺急的，没数据就没办法办公了，再说了，你一个程序员，怎么都不会恢复数据呢”。怼得我哑口无言！

经过

于是乎找了几个网上搜索方法和软件，都特么不怎么靠谱或不好使，要么让你充钱，要么让你开通会员。就一句话：现成的没有，给钱也不保证能行。

最后找到小破站的一个教学视频 《数据恢复XXOO》看了FAT那段就知道是怎么回事儿了，现在的U盘大部分都是NTFS，果断看NTFS那段，原来只是文件目录被删除了，元数据还在，这也就能讲得通有数据恢复一说了，要都是格式化删除，那都不用玩了。这个视频主要就是讲各种格式的数据存储的方式，捋一遍之后基本就啥都明白了，恢复呢？看标题：R-Studio

抱着学习的目的，在网上找了一圈之后找到了绿色的学习版，果断拿过来试一下
1.打开U盘直接删除某个文件
2.打开软件，选中硬盘，分区搜索
3.选择盘符，显示文件，右边就列出了已经删除的文件了
4.恢复，存到另一个盘

结果

R-Studio恢复数据可以说非常的傻瓜，小白都能操作，可能对于小白来说最大的难题不是使用这个软件，而是怎么找到一个可用的，不要充钱也能搞定的方法。

因此有了该文章记录，并决定教会猴哥这个软件的使用。
