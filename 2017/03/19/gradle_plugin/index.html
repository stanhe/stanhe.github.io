什么是 gradle plugin

gradle 插件指我们的build.gradle文件中的： apply plugin: ‘your_plugin’，自定义的plugin能够完成我们想要的功能，在不同的project中进行快捷的引用。

编写一个gradle plugin。

作为示例，这里编写一个Android中 常用的versionCode自增插件，目的在于我们打包的时候版本自增。
